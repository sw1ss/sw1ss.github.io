Using the following recursive algorithm we can find a way throuh the maze. It won't necessarily find the shortest path but that doesn't matter.

```
function findpath(x, y):
	if reached end point:
		return True
	if maze[x][y] is wall:
		return False
	if have already visited (x,y):
		return False

	mark (x,y) as visited

	if findpath(x+1,y) == True:
		return True
	if findpath(x-1,y) == True:
		return True
	if findpath(x,y+1) == True:
		return True
	if findpath(x,y-1) == True:
		return True

	unmark (x,y) as no longer visited

	return False

```
